Tiny\+CThread is a minimal, portable implementation of basic threading classes for C.

They closely mimic the functionality and naming of the C11 standard, and should be easily replaceable with the corresponding standard variants.\hypertarget{index_port_sec}{}\doxysection{\texorpdfstring{Portability}{Portability}}\label{index_port_sec}
The Win32 variant uses the native Win32 API for implementing the thread classes, while for other systems, the POSIX threads API (pthread) is used.\hypertarget{index_misc_sec}{}\doxysection{\texorpdfstring{Miscellaneous}{Miscellaneous}}\label{index_misc_sec}
