 \begin{DoxyNote}{Note}
The {\ttfamily \+\_\+\+Thread\+\_\+local} keyword is a macro that maps to the corresponding compiler directive (e.\+g. {\ttfamily \+\_\+\+\_\+declspec(thread)}). 

This directive is currently not supported on Mac OS X (it will give a compiler error), since compile-\/time TLS is not supported in the Mac OS X executable format. Also, some older versions of Min\+GW (before GCC 4.\+x) do not support this directive.
